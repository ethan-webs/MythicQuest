# Mythic Quest - Fantasy Strategy Game Landing Page

A bold, immersive landing page for a competitive fantasy strategy video game built with HTML, CSS, Bootstrap 5, and JavaScript.

## Project Overview

Mythic Quest is an immersive single-page landing page designed for a competitive fantasy strategy video game. The page features a dark, mystical theme with deep purple backgrounds and metallic gold accents, creating an epic gaming atmosphere. The landing page showcases game factions, features, media content, and pre-registration incentives to engage potential players and build anticipation for the game's release.
